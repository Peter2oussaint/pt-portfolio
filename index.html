Chronos

Chronos is a comprehensive developer tool that monitors the health and web traffic for containerized (Docker & Kubernetes) and non-containerized microservices communicated via REST APIs or gRPC, whether hosted locally or on Amazon Web Services (AWS). Use Chronos to see real-time data monitoring and receive automated notifications over Slack or email.

Intune

Intune is a music discovery app built with React that helps users build harmonically compatible playlists using music theory and ChatGPT-powered recommendations. Users can create accounts, sign in, and search for artists or filter songs by musical key. The app stores songs along with their key and BPM in a fast, searchable database. Intune is designed to support playlist building for DJs and music lovers alike.
